### PDF generator

## Description

A simple app for generate pdf file from input html text

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# development with docker-compose
$ docker-compose up

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
